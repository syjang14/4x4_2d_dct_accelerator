# 4x4_2d_dct_accelerator
Zynq-7000과 AMBA AXI4를 사용하는 4x4 discrete cosine transform 연산 가속기
AXI 인터페이스를 익히고 RTL 설계를 경험하기 위한 프로젝트입니다.

# Things used in this project
- HW
    - Zybo Z7-10 FPGA
- SW 
    - Xilinx Vivado 2020.2
    - Xilinx Vitis 2020.2
